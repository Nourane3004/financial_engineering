# financial_engineering
This project implements a multi-agent trading system (MACRO_INSIGHT, QUANT_MODELER, PATTERN_DETECTOR, RISK_MANAGER, RAG_AGENT).
A central orchestrator collects market data, distributes it to agents, aggregates their signals, and produces a final trading decision.
A Streamlit interface displays the agentsâ€™ outputs, charts, and explanations.
KEY_FEATURES:
Macro Insight Agent
Macro-economic analysis, global market trends, interest rates, inflation, news.

Quant Modeler Agent
Statistical modeling, regression, forecasting, quantitative signals.

Pattern Detector Agent
Candlestick patterns, chart formations, anomalies, divergences.

Risk Manager Agent
Volatility, position sizing, drawdown control, global risk score.

RAG Agent (Retrieval-Augmented Generation)
Retrieves external documents, news, PDFs, and integrates textual knowledge into analyses
